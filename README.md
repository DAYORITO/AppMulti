# AppMulti

Para correr el proyecto en docker, sigue estos pasos:

1. Ejecuta el siguiente comando para construir la imagen de Docker:
   ```bash
   docker build -t appmulti .
   ```
2. Una vez construida la imagen, ejecuta el contenedor con el siguiente comando:
   ```bash
   docker run -it appmulti
   ```
Con estos pasos puedes usar la aplicación dentro de un contenedor Docker e interactuar con ella a través de la terminal.
